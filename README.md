# 🤖 JARVIS - Voice Assistant App

**Ironman Style AI with Alarm, Reminder & Habit Tracker**

## Features

✅ Voice Commands - English & Hindi
✅ Alarms with custom labels
✅ Reminders - Water, Medicine, Exercise
✅ Habit Tracker - Daily activities
✅ Local Storage - Complete privacy
✅ Offline Operation - No internet needed

## Installation

1. Download APK from Releases
2. Enable Unknown Sources in Settings
3. Install and grant permissions
4. Enjoy!
