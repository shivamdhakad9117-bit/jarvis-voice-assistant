## 📋 **HOW TO USE:**

### **Step 1: Upload Updated Files**

GitHub pe upload karo:
- ✅ `main_with_ringtone.dart` (rename to `main.dart`)
- ✅ `pubspec_updated.yaml` (rename to `pubspec.yaml`)

### **Step 2: Install Dependencies**
```bash
flutter pub get
```

### **Step 3: Run App**
```bash
flutter run
```

### **Step 4: Set Alarm with Ringtone**

```
1. Alarm tab kholo
2. Alarm label likho
3. "Select Ringtone" dropdown se ringtone choose karo
4. Time select karo
5. "Set Alarm" click karo
```

### **Step 5: Alarm Trigger Hoga**

```
- Selected ringtone play hoga
- JARVIS voice bolega: "Alarm Alert for [label]"
- Dialog mein ringtone name dikhega
- "Dismiss" button press karo to ringtone band ho
```
