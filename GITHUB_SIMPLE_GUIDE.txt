### **STEP 4: GitHub Actions Setup (5 minutes)**

```
1. Repository ke "Actions" tab click karo
2. "Set up a workflow yourself" click karo
3. File ka naam: .github/workflows/build.yml

Neeche diya code copy-paste karo:
```

```yaml
name: Build APK

on:
  push:
    branches: [ main ]
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
    - uses: actions/checkout@v3
    
    - name: Setup Java
      uses: actions/setup-java@v3
      with:
        distribution: 'zulu'
        java-version: '11'
    
    - name: Setup Flutter
      uses: subosito/flutter-action@v2
      with:
        flutter-version: '3.19.0'
        channel: 'stable'
    
    - name: Get dependencies
      run: flutter pub get
    
    - name: Build APK
      run: flutter build apk --release
    
    - name: Upload APK
      uses: actions/upload-artifact@v3
      with:
        name: JARVIS.apk
        path: build/app/outputs/flutter-app-release.apk
```

```
4. "Start commit" click karo
5. "Commit new file" click karo
```
